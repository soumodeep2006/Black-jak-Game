
# 🃏 Blackjack Game (Python)

A simple and interactive **Blackjack card game** built using Python.
In this game, a player competes against a dealer, following standard Blackjack rules.

---

## 🎯 Objective

The goal of the game is to achieve a score as close to **21** as possible without exceeding it.
If your score goes above 21, you lose (Bust).

---

## ✨ Features

* 🎴 Random card generation system
* 👤 Player vs 🤖 Dealer gameplay
* 🔢 Automatic score calculation
* 🧠 Basic Blackjack rules implemented
* 🔄 Option to draw or stop cards
* ⚡ Fast and simple gameplay

---

## 🛠️ Technologies Used

* Python 🐍
* Random Module

## 🎮 Game Rules (Simple)

* Number cards = Face value

* Jack, Queen, King = 10 points

* Ace = 11 points (can be adjusted if needed)

* Player can choose:

  * "play" → take another card
  * "stop" → end turn

* Dealer plays automatically
* Closest to 21 win
